Introduction

If you have worked with MS suite tables you will notice that all you needed to do was to click there, import the table rows and columns, and start to fill in the blanks. In HTML the structure of the table literally has to be specified in a series of text that instruct the browsers on how to display your table. Not so straightforward, huh? Have no fear! With the aid of a handful of HTML elements and attributes, and of course CSS, you can create some pretty awesome tables! This article will give a brief overview of these elements and attributes to get you started on your next HTML table task. Lets proceed!


Elements

<table> <caption> <thead> <tr> <tbody> <td> <tfoot>

Name Format Description
Table element <table> This is how you introduce tables into your html code. It has opening and closing tags, between which the entire code for the table is nested.
Caption <caption> This HTML element specifies the caption (or title) of a table, and if used is ALWAYS the first child of a <table>.
Table Head <thead> This element is used to add headings to each column in the table. Data is entered here using the <tr> (table rows) element which would enclose some <th> elements.
Table row <tr> This is used to enter rows in an html document. It has opening and closing tags. Between these tags data is entered using either a <th>element (if in the <thead> section of a table) or a <td> element (table data) element, which is used to enter data rows everywhere else in the html table.
Table Body <tbody> This is where the main contents of an html table are inserted. It has opening and closing tags. Data is entered here using the <tr> (table rows) element which would enclose some <td> elements.
Table Data <td> This element defines a cell of a table that contains data
Table Data for <thead> <th> This element defines a cell that contains data in the <thead> section of a table
Table foot <tfoot> This element contains rows that summarize the contents of a table, e.g, totals.


Attributes

"colspan" "rowspan"

Name Format Description
Column Span colspan="" This attribute is used to extend the column space occupied by a data cell. An integar value to represent how many columns to span over is added between the parenthesis. See the example in the last row of this table, where <tfoot>and "colspan" have been used to create a data cell the occupies 3 columns.
Row Span rowspan="" This attribute is used to extend the row space occupied by a data cell. An integar value to represent how many rows to span over is added between the parenthesis
